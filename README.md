# 知网 AI 分析包助手

这是一个 Chrome / Edge 浏览器插件的第一版 MVP。它不替代知网检索，也不调用 AI API；它只做一件事：把知网检索结果页中的论文题录整理成结构化材料，并导出为可以上传到 ChatGPT、DeepSeek 等网页 AI 工具的分析包。

## 当前能力

- 在用户已经登录知网的浏览器页面中运行。
- 从知网检索结果页采集当前页可见论文。
- 保存采集进度，重复采集同一论文时自动去重。
- 导出 zip 分析包，包含：
  - `papers.csv`：表格版题录。
  - `papers.jsonl`：结构化原始数据。
  - `papers_for_ai.md`：适合直接给 AI 阅读的总文件。
  - `batch_001.md` 等：分批 Markdown，适合上千篇论文分批上传。
  - `analysis_prompt.md`：单批分析提示词。
  - `merge_prompt.md`：多批结果合并提示词。

## 使用方式

1. 安装依赖：`npm.cmd install`
2. 构建插件：`npm.cmd run build`
3. 打开 Chrome 或 Edge 的扩展管理页。
4. 开启“开发者模式”。
5. 选择“加载已解压的扩展”，目录选择本项目的 `dist` 文件夹。
6. 在浏览器中打开知网并登录。
7. 使用知网自己的检索功能得到论文列表页。
8. 点击浏览器右上角插件图标，选择“采集当前页”。
9. 需要更多论文时，手动翻到下一页后继续点击“采集当前页”。
10. 采集完后点击“导出 AI 分析包”。
11. 将 `analysis_prompt.md` 和 `batch_*.md` 上传到 ChatGPT 或 DeepSeek 分批分析，再用 `merge_prompt.md` 合并多批结果。

## 边界说明

- 插件不保存、读取或代填知网账号密码。
- 插件不绕过验证码、登录限制、付费限制或下载限制。
- 插件不批量下载 PDF / CAJ 全文。
- 第一版不做本地工作台，也不直接连接 ChatGPT / DeepSeek API。
- 当前弹窗采集按钮主要面向“检索结果页当前页”；详情页摘要解析器已经有基础实现，后续可以继续接入自动补摘要流程。

## 开发命令

- `npm.cmd test`：运行单元测试。
- `npm.cmd run build`：构建可加载的扩展目录。

在 PowerShell 中请优先使用 `npm.cmd`，避免系统执行策略拦截 `npm.ps1`。
